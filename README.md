# C# C++ invoke sample

c# c++ c++/cli invoke sample

## What's that?

+ C# exe -> C++/CLI dll -> Native C++ dll
+ C++/CLI exe -> C++/CLI dll -> Native C++ dll
+ Native C++ exe -> Native C++ dll

## Prerequisites

+ Visual Studio 2017 with C#/Native C++/C++-CLI support
